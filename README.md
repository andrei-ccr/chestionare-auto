- Categoriile B, C si D

- Stil DRPCIV

## Configurare

1. Importa baza de date din */database/chestionare.sql*
2. Modifica fisierul *connect.php* cu datele de conectare corecte
3. Website-ul ruleaza pe versiunea de PHP 7 sau mai mica + MySQL.
